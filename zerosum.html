Zero-sum game

A zero-sum game is defined as:
mathematical representation in game theory and economic theory of a situation which involves two sides, where the result is an advantage for one side and a loss for the other. If the total gains of the participants are added up, and the total losses are subtracted, they will sum to zero.

Let's put this into simple terms.
It is a game with zero or more winners, and an equal amount of losers.

For example: Tic Tac Toe

Tic Tac Toe is a zero-sum game because you can have three results:

  • You win, opponent loses
  • You lose, opponent wins
  • Cat game (tie)

Zero-sum games can be as simple as Tic Tac Toe, or as complicated as Chess, as long as the number of winners equals the number of losers.
